Node Boilerplate
================
Boilerplate created to start a Node.JS project, with several modules and features.

## Modules
- `body-parser`
- `connect-flash`
- `connect-ensure-login`
- `cookie-parser`
- `ejs`
- `express`
- `mysql`
- `passport`
- `passport-local`
- `sequelize`
- `sequelize-session`
- `sha1`
- `strftime`

## Features
- `Login`
- `Bootstrap`
