Node Boilerplate
================
Boilerplate created to start a Node.JS project, with several modules and features.

## Modules
- `body-parser`
- `cookie-parser`
- `ejs`
- `express`
- `sequelize`
- `morgan`
- `strftime`

## Features
- `Login` / `Basic user management`
- `Bootstrap`
- `JQuery`
- `Socket.io`
